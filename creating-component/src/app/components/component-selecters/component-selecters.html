componet Selecters ki 3 types hai

  1. Element
  2. Atttribute
  3. class

1.Element

  1. it is reprsented by 'selecter-name'
  2. to render used < selecter-name > < /selecter-name >
  3. it is used in tag form to renderall data of file
  4. default method hota hai ye data render krne ka

2.Attribute

  1. it is represented by '[selecter-name]'
  2. to render used < div selecter-name > </ div >
  3. Kisi element mein behavior dalna ho to

3.Class

  1. it is represent by '.selecter-name'
  2. it is used in place of class name or any Attribute
  3. to render used < div class="selecter-name" > </ div >
  4. styling k liye use kiya jata hi (rare)
